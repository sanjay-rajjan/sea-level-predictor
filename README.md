# Sea Level Predictor

Analysis of a dataset with the global average sea level change since 1880. Used the data to predict the sea level change through the year 2050.

### Data Source
Global Average Absolute Sea Level Change, 1880-2014 from the US Environmental Protection Agency using data from CSIRO, 2015; NOAA, 2015.
https://datahub.io/core/sea-level-rise
